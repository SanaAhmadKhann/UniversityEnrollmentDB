# University Enrollment Management System

## Project Description

This project implements a University Enrollment Management System using Microsoft SQL Server.

The system manages:

- Students
- Instructors
- Departments
- Courses
- Sections
- Enrollments
- Payments

## Features

- Enhanced Entity Relationship Diagram (EERD)
- Relational Schema Mapping
- SQL Server Database Implementation
- Primary and Foreign Keys
- Constraints (NOT NULL, UNIQUE, CHECK)
- Views
- Triggers
- Multi-table Joins
- Aggregate Queries
- Nested Subqueries

## Software Used

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
- GitHub
- Draw.io

## How to Run

1. Open SQL Server Management Studio.
2. Create the database.
3. Execute Database.sql.
4. Insert sample data.
5. Run queries and views.
